Door_Mat.py
-----------

specifications: 
Mat size must be N*M. ( N is an odd natural number, and M is 3 times N.)
The design will have 'WELCOME' written in the center.
The design pattern used |, . and - characters.

Sample Input 1
9 27
Sample Output 1
```sh
------------.|.------------
---------.|..|..|.---------
------.|..|..|..|..|.------
---.|..|..|..|..|..|..|.---
----------WELCOME----------
---.|..|..|..|..|..|..|.---
------.|..|..|..|..|.------
---------.|..|..|.---------
------------.|.------------
```
*********************************************
